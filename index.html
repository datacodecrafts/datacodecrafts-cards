CSS Transitions & Animations Review

hover a card to flip it and see the answer!

1Which property picks WHICH CSS properties get to transition?
transition-property
2Which property controls HOW LONG a transition takes?
transition-duration
3What does transition-timing-function: ease-in-out do?
Slow start → fast middle → slow finish
4What rule do you use to set up the steps of an animation?
@keyframes
5Which property sets how many times an animation repeats?
animation-iteration-count
6What value makes an animation loop forever?
infinite
7Which property controls how the element looks before/after it animates?
animation-fill-mode
8Big difference: transition vs. animation?
Transitions need a trigger (like :hover). Animations play on their own with @keyframes.
